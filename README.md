**Wanderlust** is a dynamic full-stack web application that simulates a travel booking platform, allowing users to explore and book campgrounds. This project demonstrates modern web development practices including server-side rendering, user authentication, and RESTful routing.

## Features

- **Dynamic Listings:** Browse campgrounds with detailed information.  
- **User Authentication & Authorization:** Secure login, signup, and route protection.  
- **CRUD Operations:** Users can create, read, update, and delete campground listings.  
- **RESTful Routing:** Clean and organized routing structure.  
- **Server-Side Rendering:** Pages rendered using EJS templates.  

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Templating:** EJS  
- **Other:** Express middleware for authentication and session management
